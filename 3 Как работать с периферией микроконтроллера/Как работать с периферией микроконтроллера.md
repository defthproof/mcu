# Задания

1. Реализовать команду get_adc(), которая будет считывать отсчёты АЦП с ножки
2. Реализовать команду get_temp(), которая будет считывать температуру микросхемы
3. Реализовать команду start_tm() и stop_tm(), которая будет выводить в COM, данные в формате uptime, adc, temp
4. Собирать эти данные при помощи Python, который при старте отправляет команду start_tm(), читает данные N-секунд, отправляет stop_tm(), строит график (код для построения красивого графика дать)
5. Запушить исходники прошивки и Python-скрипт на GitHub
